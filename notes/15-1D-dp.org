* 0. Identify DP

*When to use DP?*
If you see "max / min / number of ways" + "take or skip / include or exclude / how to partition" → likely DP.

*Three steps to solve:*
1. Define ~dp[i]~ — describe in one sentence what it represents.
2. Write the transition — ~dp[i] = f(dp[i-1], ...)~
3. Set the base case — ~dp[0] = ?~

*Two implementation styles:*

| Style     | Approach                          | When to use               |
|-----------+-----------------------------------+---------------------------|
| Top-Down  | Recursion + memo (big → small)    | Sparse state space        |
| Bottom-Up | For-loop tabulation (small → big) | Almost all 1D-DP problems |

Top-Down 模板：
#+BEGIN_SRC cpp
std::vector<int> memo(n + 1, -1);

int dp(int i) {
  if (i <= 0) return /* base case */;
  if (memo[i] != -1) return memo[i];
  // transition
  memo[i] = std::max(dp(i - 1), dp(i - 2) + nums[i]);
  return memo[i];
}
#+END_SRC

Bottom-Up 模板：
#+BEGIN_SRC cpp
std::vector<int> dp(n + 1, /* identity */);
dp[0] = /* base case */;

for (int i = 1; i <= n; ++i) {
  // transition
  dp[i] = std::max(dp[i - 1], dp[i - 2] + nums[i]);
}

return dp[n];
#+END_SRC

* 1. Primitives

DP 解題三步驟：
1. *定義 dp[i]* — 用一句話說清楚 dp[i] 是什麼
2. *寫 transition* — dp[i] = f(dp[i-1], dp[i-2], ...)
3. *設 base case* — dp[0] = ?, dp[1] = ?

* 2. Means of Combination & Abstraction
* 3. Classic Problems

| Pattern          | dp[i] 定義         | Transition                                        | Base Case                                   |
|------------------+--------------------+---------------------------------------------------+---------------------------------------------|
| Fibonacci        | 到第 i 階的方法數    | ~dp[i] = dp[i-1] + dp[i-2]~                       | ~dp[0]=0, dp[1]=1~                          |
| Take or Skip     | 考慮 [0..i] 的最佳  | ~dp[i] = max(dp[i-1], dp[i-2] + nums[i])~         | ~dp[0]=nums[0], dp[1]=max(nums[0],nums[1])~ |
| Reach Target     | 湊出金額 i 的最少次數 | ~dp[i] = min(dp[i-coin] + 1) for each coin~       | ~dp[0]=0~, 其餘 ~INT_MAX~                   |
| Kadane's         | 以 i 結尾的最大連續和 | ~dp[i] = max(nums[i], dp[i-1] + nums[i])~         | ~dp[0]=nums[0]~                             |
| Subsequence      | 以 i 結尾的 LIS 長度 | ~dp[i] = max(dp[j]+1)~ for ~j<i, nums[j]<nums[i]~ | ~dp[i]=1 for all i~                         |
| Partition/Decode | s[0..i-1] 能否拆分  | ~dp[i] = any(dp[j] && s[j..i] valid)~             | ~dp[0]=true~                                |
| Palindrome       | 以 i 為中心展開      | 從每個 center 往兩邊 expand                         | 每個字元本身是回文                             |
