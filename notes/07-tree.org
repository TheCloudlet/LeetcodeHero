** 2.1 Traversal
*** 2.1.1 DFS
Depth-first search (DFS) ::
Process the current node, recur into subtrees, return when reaching Empty.

Order of handling for each nodes:

1. In-order :: left -> mid -> right
2. Pre-order :: mid -> left -> right
3. Post-order :: left -> right -> mid

Very elegant haskell way to implement the traversal.

#+NAME: Haskell Tree Traversal
#+BEGIN_SRC haskell
-- DFS traversals
inorder :: Tree a -> [a]
inorder Empty        = []
inorder (Node x l r) = inorder l ++ [x] ++ inorder r

preorder :: Tree a -> [a]
preorder Empty        = []
preorder (Node x l r) = [x] ++ preorder l ++ preorder r

postorder :: Tree a -> [a]
postorder Empty        = []
postorder (Node x l r) = postorder l ++ postorder r ++ [x]
#+END_SRC

cpp way to implement the traversal.

#+NAME: LC94/144/145 Binary Tree Inorder/Preorder/Postorder Traversal
#+BEGIN_SRC cpp
// Recursive
void preorder(TreeNode *root) {
  if (!root) return;
  // visit root->val here (pre)
  preorder(root->left);
  preorder(root->right);
}

void inorder(TreeNode *root) {
  if (!root) return;
  inorder(root->left);
  // visit root->val here (in)
  inorder(root->right);
}

void postorder(TreeNode *root) {
  if (!root) return;
  postorder(root->left);
  postorder(root->right);
  // visit root->val here (post)
}
#+END_SRC

- Pre-order :: Stack; handle current node first, then push right then left (LIFO reverses order)
- In-order :: Stack + pointer; go left until null, pop, handle current node, then go right
- Post-order :: Modified pre-order (push left then right), reverse result at end

*** 2.1.2 BFS

Breadth-first search (BFS) ::
Process the current frontier, collect children as the new frontier, recur until empty.

Basic BFS:

#+NAME: Cpp Tree Traversal (BFS)
#+BEGIN_SRC cpp
// Recursive
void bfs(TreeNode *root, std::queue<TreeNode *> &nexts) {
  if (!root) return;

  // Visit value here // mid
  if (root->left) nexts.push(root->left);
  if (root->right) nexts.push(root->right);

  if (!nexts.empty()) {
    TreeNode *next = nexts.front();
    nexts.pop();
    bfs(next, nexts);
  }
}
#+END_SRC

Tree Level Order Traversal:

(This is the same algorithm handling expanding graphs)

#+NAME: LC102 Binary Tree Level Order Traversal
#+BEGIN_SRC cpp
// Iterative (Preferred)
std::vector<std::vector<int>> levelOrder(TreeNode *root) {
  if (!root) return {};

  std::queue<TreeNode *> nexts;
  std::vector<std::vector<int>> ans;
  nexts.push(root);

  while (!nexts.empty()) {
    int level_count = nexts.size();  // Key of level order traversal
    std::vector<int> level;
    for (int i = 0; i < level_count; ++i) {
      TreeNode *front = nexts.front();
      nexts.pop();
      level.push_back(front->val);

      // Add children to nexts
      if (front->left) nexts.push(front->left);
      if (front->right) nexts.push(front->right);
    }
    ans.emplace_back(std::move(level));
  }

  return ans;
}
#+END_SRC

** Arm's-length Recursion

Trust the next stack frame to handle nullptr — don't peek ahead.

| Context        | Action                         | Why                         |
|----------------+--------------------------------+-----------------------------|
| Tree Recursion | Only check ~if (!root)~        | Callee handles nullptr      |
| List Iteration | Check ~curr->next~ before move | No stack frame to catch you |

#+NAME: 101. Symmetric Tree
#+BEGIN_SRC cpp
// LC101. Symmetric Tree
// BAD: Dereferencing child without null check -> segfault
if (a->left->val != b->right->val) {
  return false;
}

// GOOD: Let recursion handle it
if (a->val != b->val) {
  return false;
}

return isMirror(a->left, b->right) && isMirror(a->right, b->left);
